# Usage
 
投稿機能
・名前、コメント、パスワードを入力し送信を押すと投稿が保存され、下に投稿が表示されます。

編集機能
・編集したい投稿番号を選択し、編集後の内容と該当する投稿のパスワードを入力。
・投稿内容が更新され、表示欄上部に表示されます。

削除機能
・削除したい投稿番号を選択し、該当パスワードを入力。
・投稿がデータベースから削除されます。
